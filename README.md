# AI Tetris Pro 🎮

Nowoczesna implementacja gry Tetris w czystym JavaScript, HTML i CSS.

## 🎮 Jak uruchomić

Nie wymaga instalacji! Po prostu otwórz plik `tetris.html` w przeglądarce:

```bash
# Linux
xdg-open tetris.html

# macOS
open tetris.html

# Windows
start tetris.html
```

Lub przeciągnij plik do okna przeglądarki.

## ✨ Funkcje

- **Klasyczna rozgrywka Tetris** - wszystkie oryginalne mechaniki
- **System punktacji** - śledzenie wyniku, linii i poziomu
- **Podgląd następnego klocka** - planuj swoje ruchy
- **Kontrola prędkości** - dostosuj tempo gry
- **Efekty wizualne** - ciemny motyw z neonowymi akcentami
- **Responsywny interfejs** - działa na różnych rozmiarach ekranu

## 🕹️ Sterowanie

| Klawisz | Akcja |
|---------|-------|
| ← (strzałka w lewo) | Przesuń w lewo |
| → (strzałka w prawo) | Przesuń w prawo |
| ↑ (strzałka w górę) | Obrót |
| ↓ (strzałka w dół) | Szybskie opadanie |
| Spacja | Natychmiastowy drop |
| P | Pauza |

## 🛠️ Technologie

- **HTML5** - struktura i canvas
- **CSS3** - stylowanie i animacje
- **JavaScript (ES6+)** - logika gry

Zero zależności! Cała gra w jednym pliku HTML.
